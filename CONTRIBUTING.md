# Beiträge

Vielen Dank für das Interesse an EEDTOY.

## Fehler melden

Verwende die Bug-Report-Vorlage und gib EEDTOY-Version, Windows-Version, Gateway-Typ und reproduzierbare Schritte an. Entferne persönliche Daten und sensible Geräteinformationen aus Logs und Projektdateien.

## Änderungen einreichen

1. Eigenen Branch erstellen.
2. Änderungen klein und fachlich abgegrenzt halten.
3. `npm ci` ausführen.
4. Mindestens `npm run build` erfolgreich ausführen.
5. Bei Änderungen an Installer, Electron oder Python zusätzlich den Windows-Installer testen.
6. Pull Request mit verständlicher Beschreibung und Testnachweisen erstellen.

Durch einen Beitrag wird keine Änderung der proprietären Projektlizenz vereinbart. Beiträge dürfen nur eingereicht werden, wenn der Einreichende die erforderlichen Rechte daran besitzt.
